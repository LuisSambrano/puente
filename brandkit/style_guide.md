# Puente Brand Kit & Design System

## 1. Identidad Visual

- **Logo:** Puente (Minimalist Bridge Icon).
- **Vibe:** Serio, Moderno, Persuasivo, "Ultra-Clean Fintech".

## 2. Paleta de Colores

| Token               | Hex       | Uso Principal                                        |
| :------------------ | :-------- | :--------------------------------------------------- |
| **Primary Green**   | `#2ECC71` | Brand Color, Botones Primarios, Iconos activos.      |
| **Secondary Green** | `#27AE60` | Hover states, degradados, bordes.                    |
| **Puente Black**    | `#000000` | Fondos oscuros, Textos de alto impacto.              |
| **Neutral Gray**    | `#95A5A6` | Textos secundarios, iconos inactivos.                |
| **Light Gray**      | `#ECF0F1` | Fondos de tarjetas, inputs, superficies secundarias. |
| **White**           | `#FFFFFF` | Fondo principal (Clean UI).                          |

## 3. Tipografía

- **Familia:** `Inter` (Google Fonts).
- **Pesos:**
  - **Bold (700):** Headlines y Montos grandes.
  - **Semi-Bold (600):** Subtítulos y Botones.
  - **Regular (400):** Cuerpo de texto.

## 4. UI Tokens

- **Border Radius:** `Rounded-lg` (8px-12px) para componentes pequeños, `Rounded-xl` o `2xl` para tarjetas.
- **Shadows:** Suaves y difuminadas (`shadow-lg` en Tailwind).
- **Glassmorphism:** Fondo semi-transparente con blur (`backdrop-blur-md bg-white/30`) para elementos sobre fondos complejos.
- **Iconografía:** Lucide React (Stroke width: 2px).
