# MiniPay Reference

> "Transfer globally in seconds using just a phone number. No fees. Instantly buy USDT, USDC, and cUSD with local currency." - [minipay.to](https://minipay.to)

## Developer Resources

MiniPay is built on the Celo blockchain and integrated into the Opera Mini browser. It allows developers to build "Mini Apps" that access Celo's stablecoin infrastructure.

- **Developer Documentation:** [Build on MiniPay (Celo Docs)](https://docs.celo.org/developer/build-on-minipay/overview)
- **Official Website:** [minipay.to](https://minipay.to)

## Key Concepts

- **Mini Apps:** Lightweight dApps that run inside MiniPay.
- **Stablecoins:** Focus on cUSD, USDC, USDT for payments.
- **Integration:** Uses standard Web3 providers (like ensuring `window.ethereum` or standard connection kits work within the Opera Mini context).
