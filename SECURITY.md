## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please report it via email to **security@puente.finance** (simulated).

**DO NOT** create a public GitHub issue for security vulnerabilities.

### Process

1. You email us the details.
2. We acknowledge receipt within 48 hours.
3. We send a verified fix timeline.
4. Once fixed, we publicly acknowledge your contribution.
